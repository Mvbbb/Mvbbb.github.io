P1162 填涂颜色

https://www.luogu.com.cn/problem/P1162

使用 bfs, 要点: 对于某个 0, 只要他不属于边界 0 的连通量内, 他就可以变成 2
我的写法:
先 bfs 遍历边界 0, 使用标记数组标记连通分量内的 0.
在遍历非边界位置, 遇到 0, 判断他是否已经被标记数组标记, 没有的话开始从这儿搜索,
第二种思路和第一种思路类似: 将所有的 0 变成 2, 之后将边界的 2 变为 0, 之后我们可以额外在外面加一圈 2, 这样边界的 2 就会形成一个连通分量, 减少查询次数

写出上面这段代码, 再看看别人的, 我才知道自己有多蠢, 第一次搜索把 0 变成 2 简直是弱智操作, 直接在读入的时候把 0 变成 2 不就行了吗.
